auto PtrSimplex::initiateFeasible ( ) -> bool
inline

simplex.initiateFeasible() -> bool returns true if infeasible, false if feasible The approach is to first put the equalities into HNF then, all diagonal elements are basic variables. For each non-diagonal element, we need to add an augment variable Then we try to set all augment variables to 0. If we fail, it is infeasible. If we succeed, then the problem is feasible, and we're in canonical form.
