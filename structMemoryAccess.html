◆ indexMatrix()

auto MemoryAccess::indexMatrix ( ) -> MutPtrMatrix<int64_t>
inline

indexMatrix() -> getNumLoops() x arrayDim() Maps loop indVars to array indices Letting i be the indVars and d the indices: indexMatrix()' * i == d e.g. indVars = [i, j] and indexMatrix = [ 1 1; 0 1] corresponds to A[i, i + j] Note that [i, j] refers to loops in innermost -> outermost order, i.e. for (j : J) for (i : I) A[i, i + j]
