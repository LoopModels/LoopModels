Detailed Description

Tableau for the Simplex algorithm. We need a core Simplex type that is unmanaged then for convenience, it would be nice to manage it. Ideally, we could have a type hierarchy of unmanaged -> managed with some API to make the managed generic. We also want the managed to be automatically demotable to unmanaged, to avoid unnecessary specialization.

Slack variables are sorted first.
