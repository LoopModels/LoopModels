Detailed Description

Represents a memory access that has been rotated according to some affine transform. ScheduledNode Represents a set of memory accesses that are optimized together in the LP. These instructions are all connected directly by through registers. E.g., A[i] = B[i] + C[i] is a single node because we load from B[i] and C[i] into registers, compute, and A[i];
