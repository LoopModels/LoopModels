constexpr auto Intersection::compactUnion ( Intersection  other) const -> std::variant<std::monostate, Intersection, std::pair<Intersection, Intersection>>
inlineconstexpr

if the union between this and other can be expressed as an intersection of their constituents, return that intersection. Return an empty optional otherwise. The cases we handle are: (a & b) | a = a (a & b) | (a & !b) = a
