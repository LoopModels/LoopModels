◆ getReductionDst()

constexpr auto IR::Value::getReductionDst ( ) const -> Instruction *
inlineconstexpr

Defines a cycle of instructions corresponding to a reduction e.g. x = a[i]; y = foo(x); z = bar(y); a[i] = z; The cycle would let us visit foo(x) and bar(y).
