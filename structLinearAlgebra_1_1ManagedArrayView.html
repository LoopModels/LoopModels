Detailed Description

template<class T, class S, class A = std::allocator<T>, std::unsigned_integral U = default_capacity_type_t<S>>
struct LinearAlgebra::ManagedArrayView< T, S, A, U >

Non-owning view of a managed array, capable of reallocating, etc. It does not own memory. Mostly, it serves to drop the inlined stack capacity of the ManagedArray from the type.
