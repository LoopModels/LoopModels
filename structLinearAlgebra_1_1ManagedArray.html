Detailed Description

template<class T, class S, size_t N = PreAllocStorage<T>(), class A = std::allocator<T>, std::unsigned_integral U = default_capacity_type_t<S>>
struct LinearAlgebra::ManagedArray< T, S, N, A, U >

Stores memory, then pointer. Thus struct's alignment determines initial alignment of the stack memory. Information related to size is then grouped next to the pointer.
